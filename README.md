## ✨ Features

- 🔐 User authentication (login & register)
- 🚘 Create and publish trips
- 🔍 Search trips by city and date
- 📅 Booking system
- ⭐ Feedback and rating system
- 👤 Session-based user state

---

## 🛠️ Tech Stack

**Frontend**
- React (Vite)
- JavaScript (inline styles)

**Backend**
- Node.js
- Express

**Database**
- SQLite3

---

## 🚀 Getting Started

```bash
# Clone repository
git clone https://github.com/your-username/smartcity-carpooling.git
cd smartcity-carpooling

# Start backend
cd db
npm install
node server.js

# Backend running on http://localhost:3000

# Start frontend (open a new terminal)
cd frontend
npm install
npm run dev

# Frontend running on http://localhost:5173
